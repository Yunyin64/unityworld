```
│  EXCLUDE   排除匹配   节点不能包含与Card冲突的Tag                │
│  WEIGHTED  权重匹配   Tag重合越多，被随机到的权重越高            │
│  FREE      自由匹配   不受Card Tag约束（完全随机）               │
└─────────────────────────────────────────────────────────────────┘

         匹配度 (0.0 ~ 1.0)
         │
    0.0  ├── 完全自由，Card Tag不影响节点筛选
         │
    0.5  ├── 倾向匹配，但允许偏题节点混入
         │
    1.0  └── 必须严格符合Card Tag主题
```

## TagMatcher 算法草图

```
score(query: List<Tag>, candidate: List<Tag>) → float:
    intersection = |query ∩ candidate|
    
    STRICT:   intersection == |query| ? 1.0 : 0.0
    INCLUDE:  intersection / |query|              // 覆盖率
    WEIGHTED: intersection / |query ∪ candidate|  // Jaccard相似度
    FREE:     1.0

weight = score ^ (1 / (1 - matchDegree + ε))
  matchDegree → 1.0 时，低分节点权重趋近于0（严格）
  matchDegree → 0.0 时，权重趋于均匀（自由）
```

## 匹配算法升级：基于 Multiset 的交集

```
query     = ["火","火","火","DoT"]        → {火:3, DoT:1}
candidate = ["火","燃烧","异常","DoT"]    → {火:1, 燃烧:1, 异常:1, DoT:1}

Multiset交集 = min(count_q, count_c) for each tag
             = {火:min(3,1), DoT:min(1,1)}
             = {火:1, DoT:1}
             = 交集总数 2

candidate = ["火","火","燃烧","DoT"]     → {火:2, 燃烧:1, DoT:1}
Multiset交集 = {火:min(3,2), DoT:min(1,1)}
             = {火:2, DoT:1}
             = 交集总数 3  ← 更高！更匹配！
```

## 完整的 TagBag Matcher

```
┌─────────────────────────────────────────────────────────────────┐
│  TagBagMatcher                                                  │
│                                                                 │
│  query:     Multiset<Tag>   ("我想要什么")                      │
│  candidate: Multiset<Tag>   ("我是什么")                        │
│                                                                 │
│  intersectionCount = Σ min(query[t], candidate[t]) for t in T  │
│  queryTotal        = Σ query[t]                                 │
│  candidateTotal    = Σ candidate[t]                             │
│                                                                 │
│  INCLUDE:   intersectionCount / queryTotal                      │
│  JACCARD:   intersectionCount / (queryTotal+candidateTotal      │
│                                  -intersectionCount)            │
│                                                                 │
│  rawScore ∈ [0, 1]                                             │
│  weight = rawScore ^ (1/(1-matchDegree+ε))    ← matchDegree调节│
└─────────────────────────────────────────────────────────────────┘
```
