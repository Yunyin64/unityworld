
# 卡牌系统设计哲学
### 从"使用世界"到"书写世界"

> *"卡牌是术，道途是学说，Tag 是学说的语义骨架。"*

---

## 一、核心愿景

这套卡牌系统的目标，不是提供一套可以抽卡、构筑、对战的玩法模块，而是构造一个由**叙事体验、创造体验、战斗体验**三者相互证明的循环系统。

玩家不会一开始就拥有创造权。他们先作为世界的普通修行者，在探索、战斗、阅读古籍、师从高人、触发事件的过程中，逐渐接触和收集这个世界中已有的术法与道途。随着认知提升，他们会在某个时刻发现一件事：

**世界底层并非由孤立的卡牌构成，而是由一套可以被理解、推演、组合的语义体系构成。**

卡牌只是这套语义体系的显性表现。而掌握了这套语义体系，就相当于掌握了这个世界的造物规律。

从这一刻起，玩家不再是内容消费者，而是内容创造者。

---

## 二、三个体验系统的循环

这三个体验不是并列关系，而是彼此"证明"的闭环：

```
叙事体验
  │  展示世界中已存在的道途，教化玩家理解语义规律
  │  剧情事件不是背景板，而是"已走通的语义案例库"
  ▼
创造体验
  │  玩家掌握语义后开始造术，构筑自己的修行哲学
  │  创造结果放入世界，成为这个世界真实存在的一部分
  ▼
战斗体验
  │  战斗验证创造，是玩家理论的实验场
  │  战斗结果反过来改变世界叙事，NPC与事件开始回应你
  ▼
回到叙事体验（世界承认你，你的道途被记录）
```

**关键原则：叙事证明创造，创造证明战斗，战斗证明叙事。**

三者缺一不可。如果叙事与战斗割裂，创造系统就会退化成编辑器；如果战斗不反馈世界，玩家就感受不到自己的道途真实存在于这个世界中。

---

## 三、玩家成长的三个层次

整个游戏体验围绕玩家认知的阶段性觉醒展开，分为三层：

### 第一层：使用世界
玩家是**使用者**与**观察者**。

- 通过事件、遗迹、门派获得现成卡牌
- 学习燃烧、铸造、召唤、反击等战斗概念
- 认识现有流派，感受它们的气质差异
- 体验的核心是："这个世界里真的有人修这些道。"

在这个阶段，卡牌首先是**文化存在**，然后才是功能存在。玩家不需要知道 Tag 系统的存在，只需要知道"火系卡感觉很烈"、"梦境流很诡异"。

### 第二层：理解世界
玩家是**学习者**与**研究者**。

- 发现不同流派之间的共同结构
- 意识到卡牌的强度和特性并非随机，而是来自深层语义规律
- 开始主动分析：这张卡为什么长这样？这两套体系为什么会互克？

这一层是关键的认知转折。玩家从"我拿到了什么"，转向"我开始看懂为什么"。这一步决定了玩家能否进入第三层。

### 第三层：书写世界
玩家是**创造者**与**开道者**。

- 解锁造术能力，开始用 Tag 组合创造新的 Effect 与 Card
- 构筑属于自己的修行体系，开创属于自己的道途
- 将创造的内容放入世界，甚至以 Mod 形式分享给其他玩家

在这个阶段，玩家追求的不再是毕业装备，而是：

> *"我的理论是否成立？我的流派是否优雅？我的道途能不能在这个世界留下痕迹？"*

## 四、Tag 系统：世界语言的语法

**Tag 是整个系统的原点。**

它不是检索用的标签，而是这个世界规律的最小语义单位。每一个 Trigger（触发条件）、Condition（约束条件）、Action（效果动作）都拥有一组 Tag，用来描述它"是什么"。

多个节点组合成 Effect（术理片段），多个 Effect 组合成 Card（可施展的术），多个 Card 组合成 Deck（修炼体系），再往上则是道途与修行哲学。每一层的 Tag 都是下层所有 Tag 的自然汇聚。

**Tag 的浓度由组合自然涌现，无需手动配置。**

一个 Effect 内，若多个节点都带有"火"的 Tag，那么这个 Effect 的"火浓度"就会自然很高。这种浓度不是硬编码，而是语义结构的自然结果。浓度越高，说明这个术在该语义方向上越专精，也越容易吸引同类节点，形成体系的自我强化。

### 分数预算：合法性的数学表达

每个节点都带有一个分数值：

- **Trigger** 和 **Condition**：分数为零或负数。越苛刻的触发条件，负分越大，代表它为 Action 释放了更多"强度预算"。
- **Action**：分数为正数，代表效果强度消耗。
- **复杂度**：所有节点分数绝对值之和，衡量这张术法有多独特复杂。

一张卡的合法性由两个维度共同决定：**强度分不超过上限**，**复杂度匹配稀有度**。这保证了越苛刻条件触发的技能可以越强，越复杂的术法越稀有，形成天然的平衡与稀缺感。

### TagBag 匹配：驱动生成的核心算法

系统在生成时，使用一套 **TagBag 匹配算法**，输入参数为：

- `查询方`：当前 Card 或 Deck 的 Tag 列表
- `候选池`：所有可用节点的 Tag 字典
- `匹配类型`：严格匹配 / 包含匹配 / 权重匹配 / 自由匹配
- `匹配度`：0.0（自由）到 1.0（严格）的倾向参数

**同一套算法，在系统所有层复用**：节点匹配 Effect，Effect 匹配 Card，Card 匹配 Deck，Deck 匹配道途主题。这保证了系统行为的一致性与可扩展性。

## 五、创造的乐趣层次

这套系统并非只有"生成正确结果"这一种体验，而是至少提供四种不同维度的创造快乐：

| 乐趣类型 | 描述 |
|--------|------|
| **发现快乐** | 输入几个 Tag，得到一个意料之外但合乎世界逻辑的术法 |
| **发明快乐** | 玩家感觉不是在选卡，而是在发明一招、开创一派 |
| **纯度快乐** | 追求体系的 Tag 浓度纯粹，感受"道统成型"的满足感 |
| **进化快乐** | 从节点到 Effect 到 Card 到 Deck，看着自己的体系一步步成型 |

此外，系统刻意允许"失败创造"的存在。走火入魔、异变体系、冲突 Tag 产生的意外结果，都不应被视为错误，而应是通往隐藏流派的另一条道路。

## 六、大语言模型：语义骨架的血肉

Tag 系统本质是一套可计算的**语义骨架**，而大语言模型天然擅长在骨架上添加**语义血肉**。两者的分工非常清晰：

```
规则系统（Tag / Score / Match）
    负责：这个东西能不能成立？
    优势：可控、可验证、平衡闭环

大语言模型
    负责：这个东西意味着什么？
    优势：自然语言理解、命名、世界观一致性、创意建议

协作流程：
  玩家意图（自然语言）
       ↓
  大模型：解析意图 → 候选 Tag
       ↓
  规则系统：筛选、匹配、预算、组合
       ↓
  大模型：命名、描述、世界观包装
       ↓
  玩家感知：我不是在填表单，我在表达想象
```

大模型**不参与数值平衡决策**。它的职责是：意图翻译、结果解释、命名生成、世界观校验、社区内容分类。规则系统的裁决权不应被让渡给语言模型。

---

## 七、"放入世界"：创造的终极目标

玩家的创造不止步于本地存档。系统分三个层次回应玩家的创造：

1. **系统承认**：自动生成道途名、典籍记载、NPC评价，让新卡真正成为这个世界叙事的一部分
2. **世界回应**：相关事件、NPC行为、后续剧情开始引用玩家的道途，让玩家感到"这条道真实存在"
3. **社区传播**：创造结果可导出为 Mod，供其他玩家导入，让玩家成为世界内容的共同作者

最终玩家分享的，不是数值配置，而是一种**修行哲学**、一种**战斗审美**、一种**语义流派**。这是与传统卡牌游戏分享模式本质不同的东西。

---

## 八、一句话总结

> 这不是一个让玩家"玩卡牌"的系统。
> 这是一个让玩家先读懂世界的语言，再用这门语言书写新的道途的系统。
> Tag 是语言的字，Effect 是句，Card 是段，道途是一篇流传于世的文章。
